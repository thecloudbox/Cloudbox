## Quick Start

### Download MySQL ChaosBox

**Option 1: Download from TheCloudbox.io**
```bash
# Visit https://thecloudbox.io/tools/mysql-chaosbox
# Click "Download" button
# Or use curl:
curl -L https://thecloudbox.io/downloads/mysql-chaosbox-latest.tar.gz | tar xz
cd mysql-chaosbox
```

**Option 2: GitHub Release (when available)**
```bash
# Download latest release
curl -L https://github.com/thecloudbox/mysql-chaosbox/releases/latest/download/mysql-chaosbox.tar.gz | tar xz
cd mysql-chaosbox
```

**Option 3: Docker Image**
```bash
docker pull ghcr.io/thecloudbox/mysql-chaosbox:latest
```
